# 🧠 InsightOS – AI Research & Analytics Assistant

InsightOS is an AI-powered research and analytics platform designed to help users interact with documents and datasets through a simple, intuitive web interface. The platform aims to combine modern Artificial Intelligence, data analytics, and document understanding into a single workspace for researchers, students, analysts, and professionals.

The project focuses on making data exploration easier by allowing users to upload multiple file formats, preview their content, analyze datasets, generate insights, and eventually interact with their files using natural language powered by Large Language Models (LLMs).

---

# 🚀 Current Features

* 📂 Upload CSV, PDF, TXT, and DOCX files
* 📄 Automatic file preview
* 📊 CSV dataset preview with dataset statistics
* 📁 File information (name, type, size)
* 🗂 Session-based file management
* 📈 Dataset overview including rows, columns, and missing values
* ⚙ Professional multi-page Streamlit interface
* 🎨 Clean and responsive user interface

---

# 🎯 Project Vision

InsightOS is being developed as an intelligent workspace where users can upload documents and datasets, ask questions in natural language, generate interactive visualizations, receive AI-powered insights, and create professional reports—all within a single application.

The objective is to bridge the gap between traditional data analysis tools and modern AI assistants by integrating analytics, document understanding, and Retrieval-Augmented Generation (RAG).
